## Sonic 3 Title Card Mappings Generator (S3TCG)

This script allows users to easily create title cards for Sonic 3 & Knuckles and Sonic Clean Engine

Compression tools:

* Flex 2

* mdcomp

* ClownMapEd/SonMapEd

Requirements:

* Python 3.6 or newer
 
* A Terminal / Command Prompt

How To Run

```
python3 s3tcg-api.py
```
